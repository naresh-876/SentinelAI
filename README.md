# SentinelAI

AI-powered cybersecurity log analysis tool.

## Setup

1. Clone repo
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt

4. Create `.env` file:
   GROQ_API_KEY=your_key_here

5. Run backend:
   python main.py

6. Run frontend:
   python -m http.server 5500
